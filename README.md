# 🔍 Ultimate System Diagnostic

Безопасный Bash-скрипт для глубокой диагностики Linux-систем. Собирает информацию о железе, ПО, службах, логах, сети и конфигурациях в один структурированный отчёт. Подходит как для системных администраторов, так и для повседневного использования.

## 📊 Возможности

Скрипт включает **12 полноценных разделов диагностики**:

| Раздел | Что проверяется |
|--------|----------------|
| 🖥️ Общая информация | Ядро, дистрибутив, пользователи, переменные окружения |
| ⚙️ Детальное железо | CPU, RAM, материнская плата, BIOS, GPU (NVIDIA/AMD/Intel), накопители (SMART), PCI, USB, звук, сеть, датчики температур |
| 📦 Установленное ПО | APT/DPKG, Snap, Flatpak, AppImage, Python, Node.js, Rust, Java, компиляторы |
| 🔄 Службы и процессы | systemd-юниты, таймеры, сокеты, топ процессов по CPU/RAM, зомби-процессы |
| 📜 Логи и проблемы | Failed-службы, journalctl, dmesg, segfaults, OOM killer, ошибки дисков, перегрев, сетевые ошибки, core dumps |
| 💾 Хранилища и ФС | df, inodes, LVM, RAID, точки монтирования, swap |
| 🌐 Сеть и безопасность | Интерфейсы, открытые порты, firewall (UFW/iptables/nftables), AppArmor, SSH, логи входа |
| ⚙️ Конфигурация | GRUB, модули ядра, sysctl, limits, PAM |
| 👤 Пользовательские данные | Размер `/home`, SUID/SGID файлы, world-writable файлы, cron, автозагрузка |
| 💻 Специфичное оборудование | ASUS ROG утилиты, состояние батареи ноутбука |
| 🔍 Дополнительный анализ | Статистика ФС, загруженные модули, Docker/Podman, виртуализация |
| 📋 Сводка проблем | Автоматический подсчёт критических ошибок и предупреждений |

## ✅ Безопасность и стабильность

- 🔒 **Только чтение** — скрипт никогда не изменяет конфигурацию или файлы системы.
- 🛡️ **Безопасное выполнение** — все команды обернуты в обработчики ошибок.
- 🔑 **Гибкость прав** — корректно работает как от `root`, так и от обычного пользователя (недостающие данные аккуратно пропускаются).
- 🧩 **Автоопределение утилит** — проверяет наличие необходимых команд перед запуском и адаптирует вывод.

## 🚀 Быстрый запуск

```bash
# Делаем скрипт исполняемым
chmod +x ultimate_system_scan.sh

# Полный режим (рекомендуется)
sudo ./ultimate_system_scan.sh

# Базовый режим (без прав суперпользователя)
./ultimate_system_scan.sh
